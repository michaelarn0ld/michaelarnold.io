Graphs & Networks

The Traveling Salesman.

The inspiration for this project was to explore graph traversal and the infamous “Traveling Salesman Problem”. Users create a grid of nodes where there is a defined starting and ending point and the application uses an algorithm to determine the shortest path to visit all nodes. This was created using React and by implementing concepts from the BFS algorithm.

Knowledge Management

Zettelkasten.

Zet is a command line tool for creating and maintaining a personal Zettelkasten. What is a Zettelkasten you may ask? Simply put, a Zettelkasten is a flexible repository of notes and knowledge that is organized and navigated through a series of hyperlinks. If you've ever taken notes the standard way, by creating folders ahead of the notetaking, you'll know that this method does not allow for a lot of flexibility in thought. Zettelkasten solves this problem by taking away the burden of organization and naturally creates a linked network of knowledge over time. If you are interested in learning more of the theory behind Zettelkasten, check out Niklas Luhmann’s essays here.

Shell Scripts.

This project is a continuously growing collection of command line tools that are designed with portability in mind to work across shell environments.; these tools will work in bash, zsh, or any other POSIX-compliant shell. If you’re someone who works with several different shells, these scripts are a must have!
